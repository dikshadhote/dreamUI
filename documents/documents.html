Square Avatars

Buttons

Buttons are clickable elements by which user interacts with web interface and perform some action.

Default button:

To display specific category of button use class name such as btn-secondary, btn-success, btn-warning etc.

Outline button:

You believe in simplicity and hence want something which is not loud as background colors on button?

use btn-outline-* and implement the simplicity you believe in.

Disabled button:

We can make buttons look inactive by adding the disabled boolean attribute to any element.

Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped.

Responsive Image

Use responsive-img to make image width same as parent container.If you have to change the height change aspect ratio by cropping image.

Round Image

Use round-img to make images round.

Inputs

Inputs are contact points where user gives data to the website which could be store in the database for future use.

Input with label

Label Input has a label present at the top of the input field.They can also be customized to show error-message.

Use input,input-error for default and error styling respectively.

Radio inputs

Checkbox inputs
